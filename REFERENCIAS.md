# Referencias

## Tecnologías utilizadas

- [Node.js](https://nodejs.org/) — Entorno de ejecución JavaScript del lado del servidor
- [Express 5](https://expressjs.com/) — Framework web minimalista para Node.js
- [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) — Librería síncrona para trabajar con SQLite en Node.js
- [Zod](https://zod.dev/) — Librería de validación y parsing de esquemas
- [nodemon](https://nodemon.io/) — Herramienta de desarrollo para reinicio automático del servidor
- [pnpm](https://pnpm.io/) — Gestor de paquetes rápido y eficiente para Node.js

## Material del curso

- [Manz.dev Bootcamp — APIs con NodeJS](https://manz.dev/) — Slides y ejercicios del curso, base del patrón de arquitectura utilizado
- [express-example-main](https://github.com/ManzDev) — Proyecto de referencia del curso con el patrón de rutas, validación y base de datos seguido en esta tarea
